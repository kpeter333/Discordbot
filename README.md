# KelvinBot 🤖

A Discord bot built with Python by Kelvin Peter Bally.

## What it does
- Introduces itself when it comes online
- Responds to commands like hello
- More features coming soon!

## Requirements
- Python 3.x
- discord.py
- python-dotenv

## How to install
1. Clone this repository:
   git clone https://github.com/kpeter333/Discordbot

2. Navigate into the folder:
   cd your-repo

3. Install dependencies:
   pip install -r requirements.txt

4. Create a .env file and add your token:
   DISCORD_TOKEN=your_token_here

5. Run the bot:
   python gifmaker.py
